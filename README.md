buttonfootball
==============

A button football game.

Requirements:
-------------

 * CMake (>= 2.8) [link](http://www.cmake.org/)
 * SDL2 [link](http://www.libsdl.org/)
 * OpenGL

##Complie:

 * Create a 'build' directory in the root directory. Change directory to the 'build'.
 * $ cmake ..
 * $ make
 * $ make install             (as root)
 * buttonfootball
